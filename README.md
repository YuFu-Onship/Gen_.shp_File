## 描述
生成适用于AutoCAD的 .shp 形文件，用于生成任意字体任意字词的形文件

## 使用方法
1. 脚本中设置字体路径与输出路径
2. 将生成的文件直接拖入AutoCAD
3. 使用load命令加载 .shx 文件
4. 使用 shape 命令加载形，形的名称默认为 N

## Description
Generate .shp shape files for AutoCAD, allowing you to create shape files for any text in any font.

## Usage
1. Set the font path and output path in the script.
2. Drag the generated files directly into AutoCAD.
3. Use the load command to load the .shx file.
4. Use the shape command to load the shape. The default shape name is N.
